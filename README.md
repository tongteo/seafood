##  Getting Started


###  Installation

1. Clone the chat-app repository:

```sh
git clone https://github.com/tongteo/chat-app
```

2. Change to the project directory:

```sh
cd chat-app
```

3. Install the dependencies:

```sh
> npm i
```

###  Running chat-app

Use the following command to run chat-app:

```sh
> node server js
```
